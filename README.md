# ScrapOARS

## Overview

## Installation

```sh
$ git clone https://github.com/rejuvyesh/scrapoars.git
$ cd scrapoars
$ pip install -r requirements.txt
$ python3 setup.py install
```

## Usage

```sh
$ scrapoars -s sublist.p -e elinks -m htmls -t txts -j dict.json
```
